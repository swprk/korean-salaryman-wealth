## 기대 효과

### 연간 절세 환급액

| 상황 | 연 환급액 |
|---|---|
| 연봉 5,500만원 이하 + 연금저축·IRP 풀납입 | **148.5만원** |
| 위 + ISA 만기→연금이전 활용 | **198만원** |
| 맞벌이 부부 각자 풀납입 | **최대 396만원** |

### 장기 누적 자산 효과 (30년 기준)

| 시나리오 | 누적 차이 |
|---|---|
| DC형 퇴직연금 정기예금 vs ETF 운용 | **약 1~2억원** |
| 절세계좌 활용 vs 일반 계좌만 사용 | **수천만~1억원** |
| 청년형 ISA 만 34세 골든타임 활용 | **수년 누적 가입기간 차이** |

### 청년·신혼부부 대상 정책 혜택

- **청년주택드림청약통장**: 월 100만원, 이자 **연 4.5%**, 비과세 500만원
- **청년주택드림 대출**: 분양가 80%까지 **최저 2.2% 금리**
- **신생아 특례 대출**: 출산 가구 최대 **5억원, 1.6%~**
- **신혼부부 특별공급**: 맞벌이 1.6억까지 자격

> 한 가지 항목만 놓쳐도 수백만 원~수천만 원 차이가 발생할 수 있습니다.

## 적용 전후 답변 비교

같은 질문, 두 가지 답변.
(좌측은 일반 Claude의 전형적 미스 패턴이며 항상 그렇게 답한다는 뜻은 아닙니다.)

### 케이스 1 — "30대, 연봉 4,500만원, 결혼 2~3년 내. 재테크 시작은?"

| ❌ 스킬 없이 | ✅ 스킬 적용 시 |
|---|---|
| "연금저축 600 + IRP 300 풀납입하면 **148.5만원 환급**. SCHD 추천드려요." | "연봉 5,500↓ → 16.5% 공제, 결혼 자금 락업 리스크 → **ISA 중개형 우선(서민형 400만원 한도)**, 연금저축은 월 30만원만(환급 59.4만원), DC형 퇴직연금 점검, 만 34세 이하면 청년형 ISA 대기." |

**왼쪽 답변의 결함**: ① 환급액 단정 오류(실제 99/49.5만원) ② 결혼자금 락업 무시 — 중도해지 시 16.5% 토해냄 ③ 서민형 ISA 자격 누락 ④ 종목 직접 추천(유사투자자문 회색지대)

### 케이스 2 — "ISA에서 SCHD 살 수 있죠?"

| ❌ 스킬 없이 | ✅ 스킬 적용 시 |
|---|---|
| "네, 가능합니다." (환각) | "흔히 헷갈리시는 부분이에요. ISA에서는 SCHD 직매수 불가. 국내 상장 KODEX 미국배당다우존스는 가능. SCHD 직접 매수는 일반 위탁계좌에서만." |

### 케이스 3 — "1,200만원 차익 봤는데 세금 얼마?"

| ❌ 스킬 없이 | ✅ 스킬 적용 시 |
|---|---|
| "약 264만원 (1,200 × 22%)" | "**(1,200 − 250) × 22% = 209만원**. 절세 4가지: 12·1월 분할 매도(공제 두 번), 손익통산, 매수 분산, ISA로 다음 자금 이동." |

**왼쪽 답변의 결함**: 250만원 기본공제 누락, 절세 전략 0개.

## 핵심 차별점

### 1. 사용자 상황별 분기 답변

같은 연봉 5,000만원이라도 결혼이 1년 후냐 10년 후냐에 따라 절세 전략이 정반대로 바뀝니다.

| 당신의 상황 | 권장 전략 |
|---|---|
| 5년 내 큰 지출 (결혼·주택·자녀) | ISA 우선 → 연금계좌는 부분 납입 |
| 은퇴 자금만 목표 | 연금저축·IRP 풀납입 → 환급 198만원 |
| 자산 가속기 (40~50대) | 절세계좌 풀가동 + 자녀 명의 분산 |
| 은퇴 임박 (55세+) | 인출 전략 + 건강보험 피부양자 관리 |

### 2. 환급·세금 자동 계산기

복잡한 계산을 머리로 안 하셔도 됩니다.

```bash
$ python scripts/refund_calculator.py --salary 5000 --pension 600 --irp 300

📊 연금계좌 환급액 계산 결과
연봉: 5,000만원 → 세액공제율 16.5%
연금저축    600만원  →  99만원 환급
IRP 추가    300만원  →  49.5만원 환급
💰 총 환급액: 148.5만원

$ python scripts/refund_calculator.py --gain 1200

💸 해외주식 양도세 (1,200만원 차익): 209만원
💡 절세 팁: 12월/1월 분할 매도로 최대 55만원 추가 절약 가능
```

### 3. 잘못된 정보 정정

> **사용자**: "ISA에서 SCHD 직접 살 수 있다던데요?"
>
> **Claude**: 그건 흔히 헷갈리시는 부분이에요. ISA에서는 해외 직상장 ETF는 못 사고, 비슷한 카테고리의 국내 상장 ETF(KODEX 미국배당다우존스 등)는 매매 가능해요. SCHD를 직접 사려면 일반 위탁계좌에서만 가능합니다.

> **사용자**: "연금저축 600만원 다 채우면 100% 환급받는다던데요"
>
> **Claude**: 환급액은 납입액의 16.5%(또는 13.2%)예요. 600만원 납입 시 99만원(또는 79.2만원) 환급입니다. '100% 환급'은 잘못된 정보일 가능성이 높아요.

블로그·SNS의 잘못된 정보로부터 당신을 보호합니다.

### 4. 직장인 빈출 누락 영역

- 💼 **회사 DC형 퇴직연금**: 90% 이상이 운용지시를 한 번도 안 함. 30년 방치하면 ETF 운용 대비 약 2억원 손실 가능
- 🌟 **청년 특화 제도**: 만 34세 골든타임 (청년형 ISA·청년미래적금·청년주택드림)
- 👫 **부부 각자 한도**: 맞벌이는 사실상 두 배 한도, 환급 최대 396만원
- 🏥 **건강보험 피부양자 임계점**: ISA 만기 인출 시 자격 박탈 위험 (50대+ 핵심)

## 적용 대상

### 권장 사용자

- ✅ **20~30대 사회초년생** — 청년 골든타임 (만 34세 이하 특화 제도)
- ✅ **30대 결혼·주택 준비 중** — 5년 내 자금 락업 리스크 vs 환급 트레이드오프
- ✅ **40대 자산가속기** — 절세계좌 풀가동 + 자녀 교육비 + 노후 준비 균형
- ✅ **50대+ 은퇴 임박** — 연금 인출 전략 + 피부양자 자격 관리
- ✅ **맞벌이 부부** — 부부 각자 한도 활용으로 환급 두 배

### 다루지 않는 영역

- 단순 종목 추천 (*"SCHD가 좋아요"* 같은 답변은 안 함)
- 시세 조회·차트 분석
- 부동산 직접 투자 (리츠는 다룸)
- 보험 상품 추천
- 사업자·자영업자 세무 (직장인 위주)

종목 추천은 **유사투자자문업 영역에 해당할 수 있어 의도적으로 회피**합니다. 대신 ETF 카테고리와 한국 세제상 처리를 알려드립니다.

## 설치 방법

사용 환경에 따라 세 가지 방법이 있습니다. 가장 마찰이 적은 방법부터 순서대로 안내합니다.

### 방법 1. Plugin Marketplace 등록 (권장 — Claude Code · Cowork)

한 줄 명령어로 등록 → 한 줄 명령어로 설치.

```bash
# 1. 마켓플레이스 등록 (한 번만)
/plugin marketplace add swprk/korean-salaryman-wealth

# 2. 스킬 설치
/plugin install korean-salaryman-wealth@korean-salaryman-wealth
```

업데이트 시 `/plugin marketplace update korean-salaryman-wealth`로 최신 버전을 받을 수 있습니다.

### 방법 2. `.skill` 파일 업로드 (Claude.ai 웹)

1. [Releases](https://github.com/swprk/korean-salaryman-wealth/releases)에서 `korean-salaryman-wealth.skill` 다운로드
2. Claude.ai 좌측 사이드바 → **Settings** → **Capabilities**
3. **Upload skill** → 다운로드한 파일 선택
4. 활성화 토글 ON

### 방법 3. Git Clone (개발자 · 직접 수정 의도)

```bash
mkdir -p ~/.claude/skills
cd ~/.claude/skills
git clone https://github.com/swprk/korean-salaryman-wealth.git
```

### 설치 확인 — 첫 질문 던져보기

설치 후 다음 질문을 복사해 입력하면 정상 작동 여부를 즉시 확인할 수 있습니다.

```
30대 직장인이고 연봉 4,800만원이에요.
회사 DC형 퇴직연금은 한 번도 운용 안 했고, 미국 ETF 좀 사보려고 합니다.
뭐부터 해야 할까요?
```

답변에 **"5,500만원 이하 → 16.5% 공제"**, **"DC형 퇴직연금 점검"**, **"청년형 ISA"** 같은 키워드가 등장하면 스킬이 정상 활성화된 것입니다.

### 설치가 안 될 때

| 증상 | 원인 / 해결 |
|---|---|
| `/plugin marketplace add`가 인식 안 됨 | Claude Code 최신 버전(2026년 4월 이후) 필요. `claude --version` 확인 후 업데이트 |
| `.skill` 파일 업로드 시 형식 오류 | Releases에서 `.zip`이 아닌 `.skill` 확장자 파일을 받아야 함 |
| Settings 메뉴에 "Skills" 항목이 없음 | 계정의 Capabilities 권한 미활성. Anthropic 지원 페이지 참조 |
| 설치 후 한국어 자산관리 질문에 일반론만 답함 | 스킬이 비활성. Settings에서 토글 ON 확인 후 새 대화 시작 |
| Plugin이 검증 실패 | 로컬에서 `/plugin validate` 실행 후 로그 확인 |
