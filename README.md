> *"ISA, 연금저축, IRP… 다 좋다는데 뭐부터 채워야 하지?"*

> *"미국 ETF 사고 싶은데 세금 22%? 250만원 공제? 환율 처리?"*

이 스킬을 설치하면 Claude가 **당신의 연봉·자금 시점·연령에 맞춰** 정확한 의사결정을 도와줍니다.

## 🎯 무엇을 얻을 수 있는가

### 💸 즉시 받을 수 있는 환급

| 상황 | 연 환급액 |
|---|---|
| 연봉 5,500만원 이하 + 연금저축·IRP 풀납입 | **148.5만원** |
| 위 + ISA 만기→연금이전 활용 | **198만원** |
| 맞벌이 부부 각자 풀납입 | **최대 396만원** |

### 📈 30년 누적 자산 차이

| 시나리오 | 누적 차이 |
|---|---|
| DC형 퇴직연금 정기예금 vs ETF 운용 | **약 1~2억원** |
| 절세계좌 활용 vs 일반 계좌만 사용 | **수천만~1억원** |
| 청년형 ISA 만 34세 골든타임 활용 | **수년 누적 가입기간 차이** |

### 🏠 청년·신혼이 자주 놓치는 혜택

- **청년주택드림청약통장**: 월 100만원, 이자 **연 4.5%**, 비과세 500만원
- **청년주택드림 대출**: 분양가 80%까지 **최저 2.2% 금리**
- **신생아 특례 대출**: 출산 가구 최대 **5억원, 1.6%~**
- **신혼부부 특별공급**: 맞벌이 1.6억까지 자격

> 한 가지 항목만 놓쳐도 수백만 원~수천만 원 차이가 발생할 수 있습니다.

## 🆚 일반 검색 vs 이 스킬

### ❌ 일반 검색·블로그가 주는 답

> "절세를 위해서는 연금저축 600만원과 IRP 300만원을 모두 채우시는 것이 가장 좋습니다."

이 답변의 함정: 만약 당신이 3년 내 결혼·주택을 계획한다면, 위 답변대로 풀납입했을 때 **중도해지 시 받은 환급금을 사실상 다 토해냅니다** (16.5% 기타소득세).

### ✅ 이 스킬이 주는 답

```
30대 + 연봉 4,500만원 + 5년 내 결혼 가능성
→ 16.5% 공제 + 락업 리스크 둘 다 큰 케이스

1단계: 비상금·결혼자금 분리 (이번 주)
2단계: 청년주택드림청약통장 개설 (만 34세 + 무주택자)
3단계: ISA 중개형 개설 (서민형 자격 → 비과세 400만원)
4단계: 연금저축 월 30만원 부분 납입 (환급 59.4만원)
5단계: 회사 DC형 퇴직연금 점검
6단계: 청년형 ISA 출시(2026년 말) 전 일반 ISA 가입 기간 누적

이번 주 할 일: ISA 중개형 개설 + 회사 퇴직연금 운용 현황 파악
```

**같은 질문에 정반대 가까운 답.** 이게 차이입니다.

## ✨ 핵심 차별점

### 1️⃣ 일반론이 아닌 당신의 상황별 맞춤 답변

같은 연봉 5,000만원이라도 결혼이 1년 후냐 10년 후냐에 따라 절세 전략이 정반대로 바뀝니다.

| 당신의 상황 | 권장 전략 |
|---|---|
| 5년 내 큰 지출 (결혼·주택·자녀) | ISA 우선 → 연금계좌는 부분 납입 |
| 은퇴 자금만 목표 | 연금저축·IRP 풀납입 → 환급 198만원 |
| 자산 가속기 (40~50대) | 절세계좌 풀가동 + 자녀 명의 분산 |
| 은퇴 임박 (55세+) | 인출 전략 + 건강보험 피부양자 관리 |

### 2️⃣ 2026년 최신 한국 세제 반영, 모두 공식 출처 검증

블로그 정보가 1~2년 묵은 동안, 이 스킬은:

- ✅ ISA·연금저축·IRP 한도와 환급률 (2026년 기준)
- ✅ 청년형 ISA·청년미래적금 (2026년 말 출시 예정 정보)
- ✅ 청년주택드림청약통장 (월 100만원·연 4.5%)
- ✅ 신혼부부 특별공급 (맞벌이 1.6억 완화, 2024년 7월~)
- ✅ 디딤돌·버팀목·신생아 특례 대출
- ✅ 해외주식 양도세 22%·250만원 공제·**T+1 결제** (2024년 5월부터)
- ✅ 건강보험 피부양자 자격 (2025년 9월 강화)

모든 수치는 **국세청·금융위원회·금감원·국토교통부·국민건강보험공단** 공식 출처로 검증되었습니다.

### 3️⃣ 자동 환급·세금 계산기 내장

복잡한 계산을 머리로 안 하셔도 됩니다.

```bash
$ python scripts/refund_calculator.py --salary 5000 --pension 600 --irp 300

📊 연금계좌 환급액 계산 결과
연봉: 5,000만원 → 세액공제율 16.5%
연금저축    600만원  →  99만원 환급
IRP 추가    300만원  →  49.5만원 환급
💰 총 환급액: 148.5만원

$ python scripts/refund_calculator.py --gain 1200

💸 해외주식 양도세 (1,200만원 차익): 209만원
💡 절세 팁: 12월/1월 분할 매도로 최대 55만원 추가 절약 가능
```

### 4️⃣ 흔한 오해를 정중히 정정

> **사용자**: "ISA에서 SCHD 직접 살 수 있다던데요?"
>
> **Claude**: 그건 흔히 헷갈리시는 부분이에요. ISA에서는 해외 직상장 ETF는 못 사고, 비슷한 카테고리의 국내 상장 ETF(KODEX 미국배당다우존스 등)는 매매 가능해요. SCHD를 직접 사려면 일반 위탁계좌에서만 가능합니다.

> **사용자**: "연금저축 600만원 다 채우면 100% 환급받는다던데요"
>
> **Claude**: 환급액은 납입액의 16.5%(또는 13.2%)예요. 600만원 납입 시 99만원(또는 79.2만원) 환급입니다. '100% 환급'은 잘못된 정보일 가능성이 높아요.

블로그·SNS의 잘못된 정보로부터 당신을 보호합니다.

### 5️⃣ 직장인이 자주 놓치는 영역까지 커버

- 💼 **회사 DC형 퇴직연금**: 90% 이상이 운용지시를 한 번도 안 함. 30년 방치하면 ETF 운용 대비 약 2억원 손실 가능
- 🌟 **청년 특화 제도**: 만 34세 골든타임 (청년형 ISA·청년미래적금·청년주택드림)
- 👫 **부부 각자 한도**: 맞벌이는 사실상 두 배 한도, 환급 최대 396만원
- 🏥 **건강보험 피부양자 임계점**: ISA 만기 인출 시 자격 박탈 위험 (50대+ 핵심)

## 👥 누구에게 가장 효과적인가

### 💯 이 스킬이 즉시 빛을 발하는 분

✅ **20~30대 사회초년생** — 청년 골든타임 (만 34세 이하 특화 제도)
✅ **30대 결혼·주택 준비 중** — 5년 내 자금 락업 리스크 vs 환급 트레이드오프
✅ **40대 자산가속기** — 절세계좌 풀가동 + 자녀 교육비 + 노후 준비 균형
✅ **50대+ 은퇴 임박** — 연금 인출 전략 + 피부양자 자격 관리
✅ **맞벌이 부부** — 부부 각자 한도 활용으로 환급 두 배

### 🚫 이 스킬이 다루지 않는 영역

- 단순 종목 추천 (*"SCHD가 좋아요"* 같은 답변은 안 함)
- 시세 조회·차트 분석
- 부동산 직접 투자 (리츠는 다룸)
- 보험 상품 추천
- 사업자·자영업자 세무 (직장인 위주)

종목 추천은 **유사투자자문업 영역에 해당할 수 있어 의도적으로 회피**합니다. 대신 ETF 카테고리와 한국 세제상 처리를 알려드립니다.
